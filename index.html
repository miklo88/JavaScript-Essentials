    If you want to be able to open up your browser on the dom and you are using visual studio code here is a couple steps to get you going.

  1. Go to your extensions in VSCode
  2. Type in `Live Server`
  3. Download/Install `Live Server`
  4. Open your HTML file and hit command L and command O and it should prompt open. If not, you can look to your bottom right and Live Server should be there. If so click that to launch your page. If not, right click and click "Launch Live Server. "
  5. If that doesn't work. Then google it!
